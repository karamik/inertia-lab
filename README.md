## 📦 Quick Start (5 minutes)

### Requirements
- 2 devices with Bluetooth & Wi-Fi (Linux / Raspberry Pi / Android)
- (Optional) LoRa modules for 10km range

### Build & Run

```bash
git clone https://github.com/inertia-lab/spore-0
cd spore-0
cargo build --release

# Device 1 (Seeder)
sudo ./target/release/inertiad --mode seed --bluetooth --ultrasound

# Device 2 (Harvester)
sudo ./target/release/inertiad --mode harvester --bluetooth --ultrasound
